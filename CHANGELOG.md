# Changelog

All notable changes to the Speyer UI System will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2025-02-14

### Added

- Initial release of the Speyer UI System (SUI).
- Complete color token system for light and dark modes.
- Typography scale using Inter typeface.
- 8px spacing grid with six spacing tokens.
- Border radius token system.
- Component specifications for buttons, cards, inputs, and alerts.
- Color-blind friendly alert patterns with required icon + text pairing.
- High-contrast status color variants for critical alerts.
- `sui-tokens.css` — standalone CSS custom properties file.
- Interactive demo page with component library and example landing page.
- AI integration prompts (quick and extended) for AI-assisted development.
- WCAG 2.1 AA accessibility guidelines and testing checklist.
- `prefers-reduced-motion` and `prefers-color-scheme` media query support.
- MIT License.
- Contributing guidelines.
